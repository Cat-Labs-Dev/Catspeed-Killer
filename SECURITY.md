# Security Policy

## Supported Versions
Cat-Labs-Dev actively maintains the latest release of the Catspeed Framework.
Security patches are prioritized for the current main branch.

| Version | Supported       |
| ------- | --------------- |
| 1.0.x   |:White_check_mark:|
| < 1.0   | :x:             |

## Reporting a Vulnerability
I follow the principles of **Coordinated Vulnerability Disclosure
(CVD)**. If you discover a potential security flaw in the github open a public issue.
DO NOT USE THE TOOL UNDER SECURITY i will not being taken attention if u sumbit a report throught there.

## Disclosure Policy
To Protect the integrity of the network analysis tools, i ask you to not share any technical details of any vulnerability until a fix has officialy been released.
Patches will take some time to finish, this is a solo project.
